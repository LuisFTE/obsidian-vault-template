# Getting Started with Second Brain

## What This Is

The desire to build a personal knowledge system that actually works — one that captures what matters, surfaces it when needed, and evolves with you. Not just note-taking. A cognitive partner.

The stack: **Obsidian** for storage and structure, **Claude** for reasoning and synthesis, **Git** for sync and history.

## Why Now

- Notes have been scattered across apps that don't talk to each other
- Important context keeps getting lost between conversations and between days
- The system should remember things so you don't have to hold them in your head

## What's Been Set Up

- Vault cloned and folder structure initialized
- Obsidian plugins installed (Git, Dataview, Calendar, Advanced Progress Bars)
- Two scheduled agents configured: hourly digest + daily tasks
- Templates ready for daily notes, todos, projects, people, media, places

## What's Still Needed

- [ ] Fill out `_Index/Now.md` with your actual context (job, active projects, goals)
- [ ] Write your first daily life note and let the digest agent process it
- [ ] Create your first project note for something you're actively working on
- [ ] Add a person note for someone important in your life
- [ ] Update sprint fields in `Now.md` frontmatter if you work in sprints

## Tech Stack

- **Obsidian** — local-first markdown editor, the storage layer
- **Obsidian Git** — sync via GitHub (no Obsidian Sync subscription needed)
- **Dataview** — query your notes like a database; powers Dashboard and Friction Log
- **Calendar + Daily Notes** — one-click daily note creation
- **Advanced Progress Bars** — visual progress tracking in Dashboard
- **Claude Code (CCR)** — scheduled agents that digest daily notes, surface questions, find content

## Resolution Criteria

This ephemeral resolves when the system is running smoothly for 2+ weeks — daily notes are being written, the digest agent is processing them, and the vault is accumulating useful context.
